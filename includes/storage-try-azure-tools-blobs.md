> [!TIP]
> 
> **Azure Blob Storage-erőforrások kezelése a Microsoft Azure Storage Explorer használatával**
> 
> [A Microsoft Azure Tártallózó](../articles/vs-azure-tools-storage-manage-with-storage-explorer.md) egy ingyenes, különálló alkalmazás, amely lehetővé teszi túl Microsoft[Azure Blob Storage-erőforrások kezelése](../articles/vs-azure-tools-storage-explorer-blobs.md). Használatával a Microsoft Azure Tártallózó, is vizuálisan létrehozása, olvasása, frissítése, és blob tárolók és blobok törlése, valamint hozzáférés tooyour BLOB tárolók és blobok kezelése.
