Ez a minta igényel az Azure PowerShell modul verziója 5.1.1-es vagy újabb. A verzió azonosításához futtassa a következőt: ` Get-Module -ListAvailable AzureRM`. Ha telepíteni vagy frissíteni szeretne, olvassa el [az Azure PowerShell-modul telepítését](/powershell/azure/install-azurerm-ps) ismertető cikket. 

Futtatás `Login-AzureRmAccount` kapcsolat létrehozása az Azure-ral.
