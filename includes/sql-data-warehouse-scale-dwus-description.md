Növelheti vagy csökkentheti a számítási Adattárházegységek (dwu-k) módosításával. Az alapértelmezett érték a 400 dwu-k. A számítási erőforrások növelése lineárisan a dwu-k növelését. Például módosítása a dwu-k 400 800 lesz kettős a számítási erőforrásokat. 

Az aktuális DWU-beállítások listájáért lásd: a szolgáltatási szint célkitűzés a [kapacitáskorlátait szolgáltatás](../articles/sql-data-warehouse/sql-data-warehouse-service-capacity-limits.md).
