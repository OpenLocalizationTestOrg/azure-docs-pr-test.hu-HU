# <a name="troubleshoot-allocation-failures-when-you-create-restart-or-resize-windows-vms-in-azure"></a>Hozzon létre, újraindításakor vagy átméretezésekor Windows-alapú virtuális gépek Azure-ban fellépő lefoglalási hibák elhárítása
Hozzon létre egy virtuális Gépet, indítsa újra a leállított (felszabadított) virtuális gépek vagy átméretezni egy virtuális Gépet, a Microsoft Azure számítási erőforrásokat tooyour előfizetés foglal le. Alkalmanként jelenhet hibák ezeket a műveleteket--még mielőtt hello Azure előfizetési korlátozásait. Ez a cikk ismerteti a hello okok egyes hello közös pufferallokációs hibák, és lehetséges javítási javasol. hello információkat is lehetnek hasznosak, ha azt tervezi, hogy a szolgáltatások hello telepítését.
