## <a name="version-70"></a>7.0-s verzió
### <a name="azure-application-insights-trends"></a>Az Azure Application Insights trendek
Azure Application Insights egy olyan új eszköz, a Visual Studióban, amelyek segítségével elemezheti az adott idő alatt az alkalmazás működését. A kezdéshez a **Application Insights** eszköztár gombra, vagy az Application Insights keresési ablakban válassza **felfedezés Telemetriai trendek**. Emellett a a **nézet** menüben kattintson a **más Windows**, és kattintson a **Application Insights trendek**. A kezdéshez válasszon egyet az öt gyakori lekérdezés közül. A telemetriai adatok típusok, idő tartomány és egyéb tulajdonságok alapján különböző adatkészletek elemezheti. Az adatok rendellenességeket keres, válasszon egyet az anomáliadetektálási beállítások a **nézettípus** legördülő listából. Az ablak alján található szűrőbeállítások megkönnyítik a telemetria bizonyos részhalmazainak alaposabb vizsgálatát.

![Application Insights trendek](./media/app-insights-release-notes-vsix/Trends.png)

### <a name="hockeyapp-for-universal-windows-apps"></a>Univerzális Windows-alkalmazások esetében a Hockeyappra
Béta-felosztáshoz és a felhasználói visszajelzések mellett a Hockeyappra biztosít symbolicated összeomlási reporting az univerzális Windows-alkalmazásokhoz. Hajtottunk ez még egyszerűbb hozzáadása a Hockeyappra SDK: kattintson a jobb gombbal egy univerzális Windows projektre, és kattintson **Hockey alkalmazás - összeomlások Analytics engedélyezése**. Ez az SDK-t telepíti, összeomlási gyűjtemény beállítja és a felhőben Hockeyappra erőforrás látja el az alkalmazás feltöltését a Hockeyappra szolgáltatás nélkül.

Más szolgáltatások:

* Az Application Insights keresési élményt biztosít gyorsabb és intuitívabb hajtottunk. Most, idő tartomány és a részletes szűrők automatikusan vonatkoznak, jelölje ki őket.
* Is az Application Insights keresési, most már van egy lehetőség, hogy a kódot közvetlenül az a – kéréstelemetria ugorhat.
* A Hockeyappra bejelentkezés során tapasztal élmény fejlesztéseket hajtottunk.
* A diagnosztikai eszközök kivételekhez éles telemetriai információk jelennek meg.

### <a name="search-telemetry-from-local-debug-sessions"></a>Helyi hibakeresési munkamenetek keresési telemetria
Ezzel a kiadással most kereshet az Application Insights telemetria jön létre a Visual Studio hibakeresési munkamenetben. Mielőtt használhat keresési csak akkor, ha regisztrálta az alkalmazását az Application insights szolgáltatással.    * Az a **nézet** menüben kattintson a **más Windows**, és kattintson a **Application Insights keresési**.

   * A Megoldáskezelőben bontsa ki a **ApplicationInsights.config**, és kattintson a **keresési hibakeresési munkamenetben telemetriai**.
3. Ha még nem regisztrált az Application insights szolgáltatással, a keresési ablak hibakeresési munkamenetben telemetriai üzemmódban megnyitott.
4. Kattintson a **keresési** ikonra a helyi telemetriai adatok megjelenítéséhez.

(Az Application Insights keresési ablak kapcsolatos további információkért lásd: a kibocsátási megjegyzések a következő szakaszokban verziója 4.0.)

Ha fájlinformációt érhető el, lehet, hogy bizonyos elemek hivatkozásait, de a megoldás attribútuminformáció is látható lesz.

Ha a hivatkozásra kattint, a kiválasztott metódus helye lesz Ugrás a kódot. Előfordulhat, hogy a verziószám eltérőek, de a szolgáltatás mennyiségét, hogy a kód megfelelő verziója határozza meg a későbbi kibocsátásokban megtörténik.

### <a name="add-the-sdk-to-your-project-without-signing-in-to-azure"></a>Az SDK hozzáadása a projekthez az Azure-bA bejelentkezés nélkül
Jelentkezzen be Azure Application Insights csomagok hozzáadása a projekthez, vagy a már nem rendelkezik a **új projekt** párbeszédpanelen vagy a projekt helyi menüjéből. Ha bejelentkezik, az SDK telepítve, és telemetriai adatokat küldeni a portált, mint a előtt konfigurálva. Ha nem jelentkezik be, az SDK nem kerülnek be a projekt és telemetriai diagnosztikai központ létrehoz. Beállíthatja, később irányíthatók.

### <a name="device-support"></a>Eszköztámogatás
A *csatlakozás;* 2015, azt [bejelentette](https://azure.microsoft.com/blog/deep-diagnostics-for-web-apps-with-application-insights/) , hogy a mobil fejlesztők számára az eszközök-e a Hockeyappra. Hockeyappra segítségével beta buildek a tesztelők webhelyről való terjesztése, gyűjtése és elemzése a alkalmazás összes összeomlik és visszajelzés gyűjtésére közvetlenül az ügyfelektől.
Hockeyappra az alkalmazás bármelyik úgy dönt, hogy összeállítani, iOS, Android, vagy a Windows vagy Xamarin, Cordova vagy Unity többplatformos megoldás, amely legyenek platform használatát támogatja.

A jövőbeli kiadásokban az Application Insights-bővítmény, lesz bemutatása után egy több integrált élmény Hockeyappra és a Visual Studio között. Most elindíthatja a Hockeyappra egyszerűen adja hozzá a NuGet-hivatkozás. Tekintse meg a [dokumentáció](http://support.hockeyapp.net/kb/client-integration-windows-and-windows-phone) további információt.
