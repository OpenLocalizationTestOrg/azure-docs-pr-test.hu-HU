# <a name="application-insights-for-microsoft-azure-apps"></a>Az Application Insights a Microsoft Azure-alkalmazások


Problémák észlelése, problémák megoldására, és folyamatosan fejleszteni az alkalmazások. Gyorsan diagnosztizálhat bármilyen problémát az élő alkalmazással. Megismerheti, hogyan használják a felhasználók az alkalmazást.

A konfiguráció nagyon egyszerű, és perceken belül láthatja az eredményeket.

* [Azure Cloud Services alkalmazás - webes és feldolgozói szerepkörök](app-insights-cloudservices.md)
* [Alkalmazás Azure kívül](app-insights-overview.md)
