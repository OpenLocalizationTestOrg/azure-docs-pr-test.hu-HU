# <a name="web-app-on-linux-documentation"></a>A webalkalmazás a Linux-dokumentáció
Webes alkalmazás Linux rendszeren jelenleg nyilvános előzetes verziójában, és lehetővé teszi az ügyfelek a webalkalmazások futtatási Linux platformon. Ez lehetővé teszi, hogy bizonyos típusú alkalmazások jobb alkalmazáskompatibilitást, és megkönnyíti a máshová helyezik az Azure App Service szolgáltatások Linux platformon futó meglévő web Apps alkalmazások áttelepítése.

Az ügyfelek hozhatnak létre webalkalmazások Linux rendszeren belül csak egy Linux engedélyezve van az App Service-csomag és az App Service-csomag csak tárolhatja, Linux-alapú webalkalmazások. 

> Nyilvános előzetes árazása webalkalmazás Linux 50 %-át a rendszeres ár, és az ár az App Service-csomag által felhasznált számítási órák száma halving számítja ki.
> 
> Nem garantált szolgáltatási szintje a nyilvános előzetes verzió idejének ajánlja fel.
