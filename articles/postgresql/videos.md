# <a name="azure-database-for-postgresql-videos"></a>Azure-adatbázis PostgreSQL-videók
[Első lépések az új Azure-adatbázissal PostgreSQL-szolgáltatás](https://channel9.msdn.com/events/Build/2017/B8046)

<iframe src="https://channel9.msdn.com/Events/Build/2017/B8046/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

A 2017 Microsoft //Build konferencia a videóból megtudhatja, két korai bevezetésének haszna ügyfelek hogyan azok használt Azure-adatbázis PostgreSQL szolgáltatás gyorsabb újítaniuk. Azt megtudhatja, hogyan ezek át a szolgáltatásnak, és az alkalmazásfejlesztés a következő lépések ismertetik. A Microsoft tárgyalja a fő szolgáltatásainak és ismertetik, hogyan Ön mint a fejlesztői is telepítse át a meglévő alkalmazások vagy az Azure-ban a felügyelt PostgreSQL szolgáltatást használó új alkalmazások fejlesztéséhez.
