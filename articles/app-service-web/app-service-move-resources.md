# <a name="supported-move-configurations"></a>Támogatott áthelyezés konfigurációk
Hello Azure webalkalmazás-erőforrások áthelyezheti [Resource Manager áthelyezése erőforrások API](../azure-resource-manager/resource-group-move-resources.md).

Az Azure Web Apps jelenleg áthelyezés forgatókönyvek a következő hello támogatja:

* Helyezze át a hello teljes tartalma (a webes alkalmazásokat, a app service-csomagokról és a tanúsítványok) erőforráscsoport tooanother erőforráscsoportot. 
   > [!Note]
   > hello célként megadott erőforráscsoport nem ebben a forgatókönyvben Microsoft.Web erőforrásokat tartalmazhat.

* Helyezze át a egyedi web apps tooa eltérő erőforráscsoportban, azokat továbbra is az az aktuális app service-csomag (hello app service csomag marad hello régi az erőforráscsoportban) futtatása közben.
