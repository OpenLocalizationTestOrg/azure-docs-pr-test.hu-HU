# <a name="azure-security-mvp-program"></a>Az Azure biztonsági MVP Program
Microsoft legtöbb értékes szakemberek (MVP) közösségi vezetők, akik korábban bemutatott példázó kötelezettséget segítve a mások használja a Microsoft-technológiák a használatát ki. A közösségi és a Microsoft a kivételes golgotavirág valós tudásuk és technikai segítséget osztoznak.

Folyamatban, hogy a Microsoft Azure most felismeri a közösségi szakértők különleges szakértői az Azure biztonsági értesítés problémamentesen működjön. A Microsoft Azure az Azure biztonsági hozzájárulás területen MVP is Microsoft MVPs oda.

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

Bár nem javasolt egy MVP részben csatlakozás, mert ez függ a technológia és életciklusa, azt értékelje ki a feltételeket néhány online fórumok Microsoft Answers, a TechNet és az MSDN; például egy kijelölt személy hozzájárulások gyakorolt hatása wikikre és online tartalom; Konferencia- és felhasználói csoportok; podcastok, webhelyek, blogok és közösségi; és a cikkek és -könyvekkel.

Az Azure biztonsági szakértő folytatja? Tudja, aki? Majd [jelöl ki a saját magának vagy valaki más](https://mvp.microsoft.com/Nomination/nominate-an-mvp) legyen, az Azure biztonsági MVP ma!
