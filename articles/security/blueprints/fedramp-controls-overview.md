# <a name="fedramp-overview"></a>FedRAMP áttekintése

A [Szövetségi kockázat és engedélyezési felügyeleti Program (FedRAMP)](https://www.fedramp.gov), egy Egyesült államokbeli kormányzati kiterjedő program, amely a biztonsági értékelést, engedélyezési és folyamatos figyelés szabványosított megközelítését ismerteti a felhő termékeket és szolgáltatások. FedRAMP határozza meg a biztonsági módszer alapján szabványosított származó követelmény [nemzeti szabványügyi és technológiai Intézet (NIST) különleges kiadvány SP 800 53](https://csrc.nist.gov/publications/detail/sp/800-53/rev-4/final) védelem szabályozza.
