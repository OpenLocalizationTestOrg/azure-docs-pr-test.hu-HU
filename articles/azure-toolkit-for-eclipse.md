# <a name="azure-toolkit-for-eclipse"></a>Eclipse-hez készült Azure-eszközkészlet
Az Eclipse Azure eszköztára és sablonok, amelyek segítségével könnyedén, fejleszthet, teszteléséhez, alkalmazások létrehozása és telepítése az Azure az Eclipse integrált fejlesztési környezeti (IDE) használatával biztosít a. Egy nyílt forráskódú projekt, amelynek a forráskód nem érhető el a a [MIT licenccel a projekt helyről](https://github.com/microsoft/azure-tools-for-java).

> [!NOTE]
> Az IntelliJ IDE egy Azure eszközkészletét is van. További információkért lásd: [IntelliJ Azure eszköztára](azure-toolkit-for-intellij.md).
