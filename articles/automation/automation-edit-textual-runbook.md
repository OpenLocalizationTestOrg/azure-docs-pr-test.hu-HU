### <a name="to-retrieve-the-contents-of-a-runbook-using-windows-powershell"></a><span data-ttu-id="66300-184">A Windows PowerShell használatával Runbook tartalmának beolvasása</span><span class="sxs-lookup"><span data-stu-id="66300-184">To Retrieve the Contents of a Runbook Using Windows PowerShell</span></span>
<span data-ttu-id="66300-185">Az alábbi Példaparancsok szemléltetik egy runbook parancsprogramjának beolvasását, és mentse a parancsfájlt.</span><span class="sxs-lookup"><span data-stu-id="66300-185">The following sample commands show how to retrieve the script for a runbook and save it to a script file.</span></span> <span data-ttu-id="66300-186">Ebben a példában a rendszer lekéri a vázlatként megjelölt verziót.</span><span class="sxs-lookup"><span data-stu-id="66300-186">In this example, the Draft version is retrieved.</span></span> <span data-ttu-id="66300-187">Akkor is a runbook közzétett verziójának beolvasásához, bár ez a verzió nem módosítható.</span><span class="sxs-lookup"><span data-stu-id="66300-187">It is also possible to retrieve the Published version of the runbook although this version cannot be changed.</span></span>

    $automationAccountName = "MyAutomationAccount"
    $runbookName = "Sample-TestRunbook"
    $scriptPath = "c:\runbooks\Sample-TestRunbook.ps1"

    $runbookDefinition = Get-AzureAutomationRunbookDefinition -AutomationAccountName $automationAccountName -Name $runbookName -Slot Draft
    $runbookContent = $runbookDefinition.Content

    Out-File -InputObject $runbookContent -FilePath $scriptPath
