# <a name="advisor-security-recommendations"></a>Biztonsági javaslatokat biztosít

Azure Advisor egy egységes, összevont nézetének összes Azure-erőforrások javaslatokat nyújt. Az Azure Security Center biztonsági javaslatok növelheti integrálható. A biztonsági javaslatok kaphat a **biztonsági** az Advisor irányítópult fület.

![Az Advisor biztonsági gomb](./media/advisor-security-recommendations/advisor-security-tab.png)

A Security Center az Azure-erőforrások biztonsági felügyeletének átláthatóbbá és szabályozhatóbbá tételével megkönnyíti a fenyegetések megelőzését, észlelését és elhárítását. Rendszeres időközönként megvizsgálja az Azure-erőforrások biztonsági állapotát. A Security Center javaslatokat hoz létre, amikor lehetséges biztonsági réseket észlel. A javaslatok végigvezeti Önt a szükséges vezérlők konfigurálásának lépésein. 

![Az Advisor Biztonság lap](./media/advisor-security-recommendations/advisor-security-recommendations.png)

További információ a biztonsági javaslatok: [biztonsági javaslatok kezelése az Azure Security Center](https://azure.microsoft.com/en-us/documentation/articles/security-center-recommendations/).

## <a name="how-to-access-security-recommendations-in-azure-advisor"></a>Biztonsági javaslatok az Azure Advisor elérése

1. Jelentkezzen be az [Azure Portalra](https://portal.azure.com).

2. Kattintson a bal oldali ablaktáblában **további szolgáltatások**.

3. A szolgáltatás menü ablaktáblán alatt **figyelés és felügyelet**, kattintson a **Azure Advisor**.  
 Az Advisor irányítópult jelenik meg.

4. Az Advisor irányítópultján kattintson a **biztonsági** fülre.

5. Válassza ki az előfizetést, amely javaslatokat kap, és kattintson a kívánt **javaslatok beszerzése**.

> [!NOTE]
> Advisor-javaslatokra szeretne használni, először *az előfizetés regisztrálása* az Advisor szolgáltatásban. Egy előfizetés regisztrálva amikor egy *előfizetés tulajdonosának* elindítja az Advisor irányítópulton, és rákattint a **javaslatok beszerzése** gombra. Ez egy *egyszeri művelet*. Az előfizetés regisztrálása után érheti el, az Advisor-javaslatokra *tulajdonos*, *közreműködő*, vagy *olvasó* előfizetés, egy erőforráscsoport vagy egy adott erőforrás.
