# <a name="import-your-data-into-azure-machine-learning-studio-from-another-experiment"></a>Adatok importálása az Azure Machine Learning Studióba egy másik kísérletből
[!INCLUDE [import-data-into-aml-studio-selector](../../../includes/machine-learning-import-data-into-aml-studio.md)]

Mikor érdemes egy kísérlet közbenső eredményt, és azt egy másik kísérlet részeként felhasználása alkalommal lesz. Ehhez a modul adatkészletként mentéséhez:

1. Kattintson a modult, amely menti adatkészletként kimenetét.
2. Kattintson a **Dataset elmentse**.
3. Amikor a rendszer kéri, adjon meg egy nevet és egy leírást, amely lehetővé teszi a dataset könnyen azonosíthassák.
4. Kattintson a **OK** be van jelölve.

A mentés végeztével a dataset lesz elérhető, a munkaterület kísérletek belüli használatra. A megtalálja a **mentett adatkészletek** a modulpalettán lista.
