# <a name="get-to-know-your-data-science-virtual-machine"></a>Ismerkedjen meg az adatok tudományos virtuális gép

Az adatok tudományos virtuális gép (**DSVM**) tartalmaz, valamint a népszerű Adattudomány, a Machine Learning / AI eszközök és adatok platformokon. Ezek van beállítva szigorú tesztelik az Adatelemzők és fejlesztők a Microsoft, valamint a szélesebb körű Adattudomány közösségi stabilitás és általános életképességét biztosításához konfigurációkat.

A DSVM is több Kódminták és forgatókönyvek helyileg végrehajtható Jupyter notebookok formájában. Ezeket a mintákat szövegértést támogatási dokumentációja is van.
