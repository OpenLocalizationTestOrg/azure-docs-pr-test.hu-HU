## <a name="why-azure-file-storage-is-useful"></a><span data-ttu-id="c54ac-112">Miért hasznos az Azure File Storage?</span><span class="sxs-lookup"><span data-stu-id="c54ac-112">Why Azure File storage is useful</span></span>

<span data-ttu-id="c54ac-113">Az Azure File Storage segítségével lecserélheti a helyszínen vagy a felhőben tárolt Windows Server, Linux vagy NAS-alapú fájlkiszolgálókat egy operációs rendszert nem igénylő felhőalapú fájlmegosztásra.</span><span class="sxs-lookup"><span data-stu-id="c54ac-113">Azure File storage allows you to replace Windows Server, Linux, or NAS-based file servers hosted on-premises or in the cloud with an OS-free cloud file share.</span></span> <span data-ttu-id="c54ac-114">Az Azure File Storage az alábbi előnyöket biztosítja:</span><span class="sxs-lookup"><span data-stu-id="c54ac-114">Azure File storage has the following benefits:</span></span>

* <span data-ttu-id="c54ac-115">**Megosztott hozzáférés** Az Azure-fájlmegosztások támogatják az iparági szabvány SMB protokollt, ezért zökkenőmentesen lecserélheti helyszíni fájlmegosztásait Azure-fájlmegosztásokra, és nem kell aggódnia az alkalmazások kompatibilitása miatt.</span><span class="sxs-lookup"><span data-stu-id="c54ac-115">**Shared access** Azure File shares support the industry standard SMB protocol, meaning you can seamlessly replace your on-premises file shares with Azure File shares without worrying about application compatibility.</span></span> <span data-ttu-id="c54ac-116">A fájlmegosztások több gépről, alkalmazásból/példányból való elérésének lehetősége az Azure File Storage egyik jelentős előnye.</span><span class="sxs-lookup"><span data-stu-id="c54ac-116">Being able to access a file share from multiple machines and applications/instances is a significant advantage with Azure File storage.</span></span>

* <span data-ttu-id="c54ac-117">**Teljes körűen felügyelt** Az Azure-fájlmegosztások anélkül is létrehozhatók, hogy felügyelnie kellene a hardvereket vagy az operációs rendszert. Ez azt jelenti, hogy nincs szükség a kiszolgáló operációs rendszerének kritikus fontosságú frissítésekkel történő javítására vagy a hibás merevlemezek cseréjére.</span><span class="sxs-lookup"><span data-stu-id="c54ac-117">**Fully Managed** Azure File shares can be created without the need to manage hardware or an OS, which means you don't have to deal with patching the server OS with critical security upgrades or replacing faulty hard disks.</span></span>

* <span data-ttu-id="c54ac-118">**Szkripthasználat és eszköztámogatás** A PowerShell-parancsmagok és az Azure parancssori felület (CLI) használatával az Azure File-megosztásokat az Azure-alkalmazások felügyeletének részeként hozhatja létre, csatlakoztathatja és kezelheti.</span><span class="sxs-lookup"><span data-stu-id="c54ac-118">**Scripting and Tooling** PowerShell cmdlets and Azure CLI can be used to create, mount, and manage Azure File shares as part of the administration of Azure applications.</span></span> <span data-ttu-id="c54ac-119">Azure-fájlmegosztásokat az [Azure Portal](https://portal.azure.com) és az [Azure Storage Explorer](https://storageexplorer.com) használatával hozhat létre és kezelhet.</span><span class="sxs-lookup"><span data-stu-id="c54ac-119">You can create and manage Azure File shares using the [Azure portal](https://portal.azure.com) and the [Azure Storage Explorer](https://storageexplorer.com).</span></span> 

* <span data-ttu-id="c54ac-120">**Rugalmasság** Az Azure File Storage szolgáltatást a kezdetektől úgy hozták létre, hogy mindig rendelkezésre álljon.</span><span class="sxs-lookup"><span data-stu-id="c54ac-120">**Resiliency** Azure File storage has been built from the ground up to be always available.</span></span> <span data-ttu-id="c54ac-121">Ha lecseréli helyszíni fájlmegosztásait az Azure File Storage szolgáltatásra, akkor többé nem kell helyi áramkimaradásokkal vagy hálózati hibákkal megküzdenie.</span><span class="sxs-lookup"><span data-stu-id="c54ac-121">Replacing on-premises file shares with Azure File storage means you no longer have to wake up to deal with local power outages or network issues.</span></span> 

* <span data-ttu-id="c54ac-122">**Ismerős programozhatóság** Az Azure-ban futó alkalmazások [a fájlrendszer adatátviteli API-jain](https://msdn.microsoft.com/library/system.io.file.aspx) keresztül férhetnek hozzá a megosztás adataihoz.</span><span class="sxs-lookup"><span data-stu-id="c54ac-122">**Familiar Programmability** Applications running in Azure can access data on the share via [file system I/O APIs](https://msdn.microsoft.com/library/system.io.file.aspx).</span></span> <span data-ttu-id="c54ac-123">A fejlesztők épp ezért meglévő kódjaik és képességeik felhasználásával áttelepíthetik az alkalmazásokat.</span><span class="sxs-lookup"><span data-stu-id="c54ac-123">Developers can therefore leverage their existing code and skills to migrate existing applications.</span></span> <span data-ttu-id="c54ac-124">A rendszer átviteli API-jai mellett az Azure Storage bármelyik ügyfélkódtárát (például a [.NET-hez](/dotnet/api/overview/azure/storage?view=azure-dotnet) elérhetőt) vagy az [Azure Storage REST API-ját](/rest/api/storageservices/file-service-rest-api) is használhatja.</span><span class="sxs-lookup"><span data-stu-id="c54ac-124">In addition to System IO APIs, you can use any of the Azure storage client Libraries, such as the one for [.NET](/dotnet/api/overview/azure/storage?view=azure-dotnet), or the [Azure Storage REST API](/rest/api/storageservices/file-service-rest-api).</span></span>

<span data-ttu-id="c54ac-125">Az Azure-fájlmegosztások az alábbiakra használhatók:</span><span class="sxs-lookup"><span data-stu-id="c54ac-125">Azure File shares can be used to:</span></span>
