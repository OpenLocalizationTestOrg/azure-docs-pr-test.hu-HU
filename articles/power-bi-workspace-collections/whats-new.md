# <a name="whats-new-in-power-bi-workspace-collections"></a>A Power BI munkaterület gyűjtemények újdonságai

Szoftverfrissítések **Power BI munkaterület gyűjtemények** rendszeresen kiadásakor. Azonban nem minden kiadás új felhasználók számára is elérhető szolgáltatások; Néhány kiadásokban háttérszolgáltatásnak képességek összpontosítanak. A Microsoft jelölje ki az új felhasználók számára is elérhető képességek itt.

> [!IMPORTANT]
> A Power BI munkaterületi gyűjtemények szolgáltatás elavult, és 2018 júniusáig vagy a szerződésében jelzett időpontig érhető el. Javasoljuk, hogy az alkalmazása zavartalan működése érdekében tervezze meg a migrációt a Power BI Embedded szolgáltatásba. Az adatok a Power BI Embedded szolgáltatásba való migrálásának részleteiért lásd a [Power BI munkaterületi gyűjtemények tartalmának Power BI Embedded szolgáltatásba történő migrálásával](https://powerbi.microsoft.com/documentation/powerbi-developer-migrate-from-powerbi-embedded/) foglalkozó cikket.

## <a name="july-11-2016"></a>2016. július 11.
Ez a kiadás tartalmazza:

* **Nagyszerű híreket!** A Power BI munkaterület gyűjtemények szolgáltatás most már az előzetes verzió – a most GA (általánosan elérhető).  
* Az összes REST API-k helyeztünk át **/beta** való **/v1.0**.
* .NET- és JavaScript SDK-k frissült **1.0-s verziójú**.
* A Power BI API-hívások most hitelesítése közvetlenül az API-kulcsok használatával. Alkalmazási jogkivonatok csak beágyazás van szükség. Ennek részeként kiépítés és fejlesztői jogkivonatokat elavulttá vált az 1.0-s verziójú API-k, de azok lesz továbbra is működnek az béta verziójához csak 2016. December 30. További tudnivalókért lásd: [hitelesítése és engedélyezése a Power BI munkaterület gyűjtemények](app-token-flow.md).
* Sor-biztonság (RLS) támogatja az alkalmazási jogkivonatok és a beágyazott jelentések. További tudnivalókért lásd: [sor: a Power BI munkaterület gyűjtemények biztonsági](row-level-security.md).
* Frissítve az összes mintaalkalmazás **1.0-s verziójú** API-hívásokat.
* A Power BI munkaterület gyűjtemények támogatja az Azure SDK-t, a PowerShell és a parancssori felület.
* A képi megjelenítés adatainak a felhasználó exportálhatja egy **.csv**.
* A Power BI munkaterület gyűjtemények mostantól támogatja a összes azonos nyelvi/és területi beállítások szerint a Microsoft Azure. További tudnivalókért lásd: [Azure - nyelvek](http://social.technet.microsoft.com/wiki/contents/articles/4234.windows-azure-extent-of-localization.aspx).
