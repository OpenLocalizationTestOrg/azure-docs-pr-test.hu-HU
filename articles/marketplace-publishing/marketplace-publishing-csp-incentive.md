# <a name="view-and-change-your-cloud-solution-provider-reseller-incentive-in-the-azure-marketplace"></a>Megtekintése és módosítása a Cloud Solution Provider "Viszonteladóhoz célzó olyan ösztönzők előnyeivel" az Azure piactéren
2016. március 15. a megoldás szolgáltatók (CSP) "Viszonteladóhoz célzó olyan ösztönzők előnyeivel" piactér közzétevők számára elérhető lesz biztosítjuk.  Az alapértelmezett beállítás ösztönző értéke **Igen**.  Közzétevő eldöntheti, hogy lemondja a ösztönző program.  Ez a cikk bemutatja, hogyan módosítsa a publisher profil beállítását, így akkor is lemondáshoz.

Megtekintéséhez vagy a "Viszonteladóhoz ösztönzőkkel járó" állapotának módosításához kövesse az alábbi lépéseket:

1. Jelentkezzen be a [Microsoft Azure közzétételi fiók](https://publish.windowsazure.com/workspace)

   ![Profil követően képernyő][1]
2. Kattintson a **közzétevők**

   ![Profil követően képernyő - közzétevők hivatkozás][2]
3. Figyelje meg, hogy a kriptográfiai szolgáltatók a profil információkat.  Van egy **mondja ki Igen vagy nem váltani** a "viszonteladóhoz célzó olyan ösztönzők előnyeivel" - alapértelmezett állapota nem "Igen" gombra.

   ![Profil - közzétevők][3]
4. A "viszonteladóhoz célzó olyan ösztönzők előnyeivel" módosítása állapot érhető el a közzétevő profil, ezután jelölje ki, és mentse a választott:

   * Válassza ki **Igen** a 20 %-os kedvezményes adhat ki vannak továbbértékesítése, alkalmazások és szolgáltatások az Azure piactéren listázott a CSP-k számára.  Válassza ki vagy **nem** ösztönző program megadásának lehetőségét.  Mindkét esetben a közzétevő azonos kifizetés kap.
   * Győződjön meg arról, hogy a kijelölés után kattintson **mentése** jobb felső sarokban.

   ![-Közzétevők - profil módosítási mezőbe][4]
