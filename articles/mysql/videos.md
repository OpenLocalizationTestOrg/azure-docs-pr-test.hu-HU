# <a name="azure-database-for-mysql-videos"></a>Azure-adatbázis MySQL-videók

[Első lépések az új Azure-adatbázissal MySQL-szolgáltatás](https://channel9.msdn.com/events/Build/2017/B8045)

<iframe src="https://channel9.msdn.com/Events/Build/2017/B8045/player" width="800" height="450" allowFullScreen frameBorder="0"></iframe>

Ez a videó az lehet, hogy 2017 a Microsoft //Build konferencia, további tudnivalók a Microsoft által felügyelt MySQL ajánlat az Azure-ban. Azt ismerteti, a Microsoft nyílt forráskódú adatbázis rendszerek támogatásához az Azure-ban stratégiája keresztül. Mit jelent, ha Ön továbbfejlesztésével és MySQL az Azure-ban használó alkalmazások központi telepítése fejlesztőként arról lesz szó. A Microsoft a szolgáltatás architektúrájának áttekintése megjelenítése, és mutassa be, MySQL az Azure-adatbázis más Azure-szolgáltatásokon, például webalkalmazások integrálva van.
