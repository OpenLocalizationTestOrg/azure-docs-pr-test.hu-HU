# <a name="azure-database-for-mysql-documentation"></a>A MySQL-hez készült Azure Database dokumentációja

A MySQL-hez készült Azure Database egy relációsadatbázis-szolgáltatás, amely a nyílt forráskódú MySQL Server motoron alapszik.  A teljes körűen felügyelt, szolgáltatásként nyújtott adatbázis kiszámítható teljesítménnyel és dinamikus skálázhatósággal kezeli a létfontosságú számítási feladatokat. A MySQL-hez készült Azure Database-ben végzett alkalmazásfejlesztéssel kihasználhatja a nyílt forráskódú eszközöket, Ön választhatja ki a kívánt platformot. Gyors útmutatóink, oktatóanyagaink és példáink segítségével megismerheti a MySQL-hez készült Azure Database használatát.
